# Project A リリースノート

## プロジェクト名
Numpad Typer（テンキー定型文入力ツール）

## 公開URL
https://project-a-lime-delta.vercel.app

## 概要
テンキー（0-9）を押すだけで、あらかじめ設定した定型文を瞬時に入力できる業務効率化ツール。

## 主要機能
1. テンキー連動の定型文挿入機能
2. テンプレートのリアルタイム編集・自動保存（localStorage）
3. ダークモード切り替え機能

## 使用技術
- Next.js (App Router)
- Tailwind CSS
- Vercel（ホスティング）
- AI (Cursor + Claude 3.5 Sonnet)

## 開発プロセスの振り返り
- かかった時間: 約4時間
- AIにどれくらい頼ったか: 9割（エラー解消とコード生成はほぼAI）
- 苦労した点: フォルダ構成の理解と、Gitでの送信作業。
