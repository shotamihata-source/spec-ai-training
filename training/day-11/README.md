# Numpad Typer

テンキー（0-9）で定型文を本文エリアへ挿入する、1ページ完結のWebアプリです。  
Next.js App Router + Tailwind CSS で実装しています。

## セットアップ
1. `npm install`
2. `npm run dev`
3. PCのブラウザで `http://localhost:3000` を開く（別アプリが3000番を使っている場合は `npx next dev -p 3001` など）。

## スマホから同じWi‑Fiで見る
1. MacBook とスマホを **同じWi‑Fi** に接続する。
2. ターミナルで `training/day-11` に移動し、LAN 向けに起動する。  
   `npm run dev:lan`  
   （`0.0.0.0` で待ち受け、ポート **3001**）
3. Mac のローカルIPを調べる（例）:  
   `ipconfig getifaddr en0`  
   （Wi‑Fiが別インターフェースのときは `networksetup -listallhardwareaddresses` や「システム設定 → ネットワーク」で確認）
4. スマホのブラウザで `http://（手順3のIP）:3001` を開く（例: `http://192.168.1.10:3001`）。
5. 開けないときは、Mac の **ファイアウォール** が Node/ターミナルの着信をブロックしていないか確認する。

**注意:** 開発サーバーは同一LAN内に公開される状態になるため、信頼できる自宅Wi‑Fiでの利用を推奨します。

## 使い方
1. 右側の「テンキー割り当て（0-9）」で各キーの定型文を編集する。
2. 定型文の挿入: 外付けテンキー（NumLock ON）の 0–9、または **本文エリアにフォーカスした状態**でキーボード上段の 0–9、または画面上の **クイック挿入ボタン** を使う（MacBook でも利用可）。
3. 「コピー」で本文をクリップボードへコピーし、「クリア」で本文を消去する。
4. 右上ボタンでライト/ダークモードを切り替える。

## 主な仕様
- 設定はブラウザの `localStorage` に自動保存され、再読み込み後も復元される。
- テンキー押下時は対象行が一時的にハイライトされる。
- 本文内に `[会社名]` のようなプレースホルダーが残っている場合、補助メッセージを表示する。
- ダークモード設定も `localStorage` で保持する。

## 対応環境
- PCブラウザ（Chrome / Edge / Safari の最新版想定）
- スマホブラウザでも見やすいレスポンシブUI

## 簡易テスト観点
- テンキー 0-9 の入力で対応する文面が挿入される。
- 文字の途中にカーソルを置いても、その位置に挿入される。
- ページを再読み込みしてもテンプレート設定が残る。
- クリップボードの権限不足時にエラー文言が表示される。
- ダークモード設定が再読み込み後も保持される。
