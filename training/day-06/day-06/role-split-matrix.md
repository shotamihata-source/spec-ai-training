# AI / 人間 / 協働 — 役割分担マトリクス

## 判断の基準

- **AI に任せていい**: 定型作業、大量処理、初稿生成、データ集計
- **人間がやるべき**: 最終判断、感情が絡む対応、倫理的判断、関係構築
- **協働が最適**: AI で下書き → 人間がレビュー、AI で分析 → 人間が解釈

---

## マトリクス

| 業務 | AI | 人間 | 協働 | 理由 |
|------|:--:|:----:|:----:|------|
| 市場調査レポートの作成 | | | ✅ | AIがデータ収集・集計・初稿を作るのは得意だが、「自社にとって何が重要か」の解釈や戦略への落とし込みは人間の判断が必要。AIに調査と下書きを任せて、人間が考察と結論を仕上げるのがベスト。 |
| クライアントへの初回提案メール | | | ✅ | 文面の作成はAIに任せられるが、初回メールは相手との関係構築の第一歩。相手の状況や温度感に合わせた表現の調整、送るタイミングの判断は人間がやるべき。AIが下書き→人間がトーンや内容を最終調整する流れが最適。 |
| 社内会議の議事録作成 | ✅ | | | 発言内容の文字起こし・要約・整形は AI の得意分野。定型的な作業で感情や判断が絡まないため、AIにほぼ全て任せられる。録音データを渡せば自動で議事録が作れるツールも多い。 |
| 採用面接の質問設計 | | | ✅ | 「こういうポジションにはどんな質問が効果的か」のたたき台をAIに出させるのは有効。ただし、自社のカルチャーに合うか、聞いてはいけない質問（差別につながる質問等）が含まれていないかは人間がチェックする必要がある。 |
| 契約条件の最終交渉 | | ✅ | | 交渉は相手の表情・反応を読みながらリアルタイムで判断する行為。信頼関係の構築が不可欠で、感情面の配慮も必要。法的・経営的なリスク判断も含むため、人間が直接やるべき領域。 |
| 競合分析スライドの作成 | | | ✅ | 競合の情報収集・比較表の作成・スライドの初稿はAIが効率よくこなせる。ただし「自社の強みをどう見せるか」「どの切り口で比較するか」は営業戦略に関わるので人間が方向性を決めて仕上げる。 |
| チームメンバーの評価面談 | | ✅ | | 評価面談は相手のモチベーションやキャリアに直接影響する。信頼関係をベースにした対話であり、相手の感情に寄り添いながらフィードバックする必要がある。人間同士でないと成立しない業務。 |
| プロダクトのFAQ作成 | ✅ | | | よくある質問と回答のパターンは定型的で、既存の資料やドキュメントから AIが自動生成しやすい。量が多くても一括で処理できる。内容に誤りがないか軽くチェックすればほぼAI単独で完結する。 |
| 重要顧客へのクレーム対応 | | ✅ | | クレーム対応は感情のケアが最優先。相手は怒っている状態であり、共感や誠意を示すのは人間にしかできない。対応を間違えると取引停止にもつながるため、経験と判断力がある人間が直接対応すべき。 |
| ブログ記事の執筆 | | | ✅ | AIに構成案や初稿を書かせれば大幅に時短できる。ただし、自社の独自の視点・経験談・ブランドのトーンを反映させるのは人間の役割。AIの下書きをベースに、人間が自社らしさを加えて仕上げる形が最も効率的。 |

## まとめ

| 分類 | 件数 | 該当業務 |
|------|:----:|----------|
| AI が担当 | 2件 | 議事録作成、FAQ作成 |
| 人間が担当 | 3件 | 契約交渉、評価面談、クレーム対応 |
| 協働 | 5件 | 市場調査、提案メール、面接質問、競合分析、ブログ記事 |

**傾向**: 半分が「協働」になった。実務では「AIに全部任せる」か「人間だけでやる」かの二択ではなく、AIと人間が役割分担するケースが最も多い。特に「AIが初稿・下書き → 人間がレビュー・仕上げ」のパターンは、ほぼすべてのライティング業務に当てはまる。一方で、感情・信頼関係・最終判断が絡む業務は人間がやるべき領域として残る。
